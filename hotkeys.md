# HOTKEYS
> F1 — открыть/закрыть │ q — выход

## Zellij

### Режимы (Esc — выход из режима)

  Ctrl+P              Панели
  Ctrl+T              Вкладки
  Ctrl+N              Ресайз
  Ctrl+S              Скролл
  Ctrl+O              Сессия
  Ctrl+Q              Выход

### Ctrl+P — Панели

  n                   Новая панель
  d                   Панель снизу
  r                   Панель справа
  x                   Закрыть
  f                   Fullscreen
  w                   Float toggle
  z                   Рамки toggle
  h/j/k/l             Навигация

### Ctrl+T — Вкладки

  n                   Новая
  x                   Закрыть
  r                   Переименовать
  h/l                 Пред./след.
  1-5                 По номеру

### Ctrl+N — Ресайз

  h/j/k/l             Изменить размер
  +/-                 Больше/меньше

### Ctrl+S — Скролл

  j/k                 Вниз/вверх
  Ctrl+D/U            Полстраницы
  g/G                 Начало/конец

### Ctrl+O — Сессия

  d                   Detach
  q                   Выход

## Alacritty

  Ctrl+Shift+C        Копировать
  Ctrl+Shift+V        Вставить
  Ctrl+Shift+F        Поиск
  Ctrl++/-/0          Шрифт

## Claude Code

  Enter               Отправить
  Shift+Enter         Новая строка
  Ctrl+C              Прервать
  Ctrl+R              История (atuin)
  Ctrl+G              Внешний редактор
  /clear              Очистить контекст
  /compact            Сжать контекст
  /cost               Стоимость
  /model              Сменить модель

## CLI

  z <dir>             Умный cd (zoxide)
  ll                  Список файлов
  lt                  Дерево (2 уровня)
  fd <name>           Поиск файлов
  rg <pattern>        Поиск в содержимом
  bat <file>          Просмотр с подсветкой
  dust                Размер директорий
  gitui               Git TUI
  btm                 Мониторинг системы
  tldr <cmd>          Краткая справка
