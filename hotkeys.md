# HOTKEYS

## tmux (Zellij-like)

### Режимы (Esc/Enter — выход)

  Ctrl+G              Lock/Unlock
  Ctrl+P              Панели (PANE)
  Ctrl+T              Вкладки (TAB)
  Ctrl+O              Сессии (SESS)

### Ctrl+P — Панели

  d                   Панель снизу
  r                   Панель справа
  x                   Закрыть
  f                   Zoom toggle
  h/j/k/l             Навигация (или стрелки)
  e                   Выровнять горизонтально
  E                   Выровнять вертикально

### Ctrl+T — Вкладки

  n                   Новая
  x                   Закрыть
  r                   Переименовать
  h/l                 Пред./след. (или стрелки)
  Tab                 Последняя
  1-5                 По номеру

### Ctrl+O — Сессии

  d                   Detach
  w                   Список сессий
  r                   Переименовать

### Быстрые (без режима)

  Alt+1..5            Переключить/создать вкладку
  Ctrl+G              Lock (отключить хоткеи)

### Мышь

  Выделение           Копировать в буфер
  Колёсико (средняя)  Вставить из буфера
  ПКМ                 Контекстное меню tmux
  Колёсико            Скролл

### Prefix (Ctrl+B)

  |                   Сплит горизонтально
  -                   Сплит вертикально
  c                   Новое окно

### Сохранение сессии (tmux-resurrect)

  Ctrl+B → Ctrl+S     Сохранить
  Ctrl+B → Ctrl+R     Восстановить

## Alacritty

  Ctrl+Shift+C        Копировать
  Ctrl+Shift+V        Вставить
  Ctrl+Shift+F        Поиск
  Ctrl++/-/0          Шрифт

## Claude Code

  Enter               Отправить
  Shift+Enter         Новая строка
  Ctrl+C              Прервать
  Ctrl+R              История (atuin)
  /clear              Очистить контекст
  /compact            Сжать контекст
  /cost               Стоимость
  /model              Сменить модель

## CLI

  z <dir>             Умный cd (zoxide)
  ll                  Список файлов
  lt                  Дерево (2 уровня)
  fd <name>           Поиск файлов
  rg <pattern>        Поиск в содержимом
  bat <file>          Просмотр с подсветкой
  dust                Размер директорий
  gitui               Git TUI
  btm                 Мониторинг системы
  tldr <cmd>          Краткая справка
