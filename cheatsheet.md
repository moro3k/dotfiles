# ШПАРГАЛКА ПО ОКРУЖЕНИЮ
> F1 — открыть/закрыть | q — выход | Scroll — листать

═══════════════════════════════════════════════════════════════════════════════
## ZELLIJ — Мультиплексор терминала
═══════════════════════════════════════════════════════════════════════════════

### Режимы (нажми для входа, Esc — выход)
  Ctrl+P              Панели (Pane)
  Ctrl+T              Вкладки (Tab)
  Ctrl+N              Изменение размера (Resize)
  Ctrl+S              Скролл (Scroll)
  Ctrl+O              Сессия (Session)
  Ctrl+Q              Выход из Zellij

### В режиме панелей (Ctrl+P)
  n                   Новая панель
  d                   Новая панель снизу
  r                   Новая панель справа
  x                   Закрыть панель
  f                   Fullscreen (развернуть)
  w                   Toggle floating панели
  h/j/k/l             Навигация ←↓↑→

### В режиме вкладок (Ctrl+T)
  n                   Новая вкладка
  x                   Закрыть вкладку
  r                   Переименовать
  h/l                 Предыдущая/следующая
  1-5                 Перейти к вкладке по номеру

### В режиме ресайза (Ctrl+N)
  h/j/k/l             Изменить размер ←↓↑→
  +/-                 Увеличить/уменьшить

### В режиме скролла (Ctrl+S)
  j/k                 Скролл вниз/вверх
  Ctrl+D/U            Полстраницы вниз/вверх
  g/G                 В начало/конец

### Сессии
  Ctrl+O, d           Отключиться (detach)
  zellij attach       Подключиться обратно

═══════════════════════════════════════════════════════════════════════════════
## WEZTERM — Терминал (поверх Zellij)
═══════════════════════════════════════════════════════════════════════════════

### Вкладки WezTerm
  Ctrl+T              Новая вкладка
  Ctrl+W              Закрыть вкладку
  Ctrl+Tab            Следующая вкладка
  Alt+1..5            Вкладка по номеру

### Панели WezTerm (если без Zellij)
  Ctrl+Shift+D        Разделить вертикально │
  Ctrl+Alt+D          Разделить горизонтально ─
  Alt+H/J/K/L         Навигация между панелями
  Ctrl+Shift+Z        Zoom панели

### Буфер обмена и поиск
  Ctrl+Shift+C        Копировать
  Ctrl+Shift+V        Вставить
  Ctrl+Shift+F        Поиск по терминалу

### Шрифт
  Ctrl++              Увеличить
  Ctrl+-              Уменьшить
  Ctrl+0              Сбросить
  Ctrl+Scroll         Колёсиком

### Система
  Ctrl+Shift+R        Перезагрузить конфиг
  Ctrl+Shift+P        Палитра команд

## CLAUDE CODE — AI-ассистент
═══════════════════════════════════════════════════════════════════════════════

### Основные
  Enter               Отправить сообщение
  Escape              Отмена ввода
  Ctrl+C              Прервать операцию
  Ctrl+D              Выход

### Навигация
  Ctrl+R              Поиск по истории
  ↑ / ↓               История сообщений

### Дополнительно
  Ctrl+T              Показать/скрыть задачи
  Ctrl+O              Развёрнутый лог
  Ctrl+S              Сохранить промпт
  Ctrl+G              Внешний редактор

### Slash-команды
  /help               Справка
  /clear              Очистить контекст
  /compact            Сжать контекст
  /status             Статус сессии
  /cost               Стоимость
  /model              Сменить модель
  /memory             Показать память

## CLI — Rust-утилиты
═══════════════════════════════════════════════════════════════════════════════

### Файлы и навигация
  z <dir>             Умный cd (zoxide) — помнит частые пути
  ll                  Список файлов с деталями
  lt                  Дерево файлов (2 уровня)
  fd <name>           Поиск файлов по имени
  rg <pattern>        Поиск в содержимом файлов

### Просмотр
  bat <file>          Просмотр с подсветкой синтаксиса
  dust                Размер директорий (визуально)
  procs               Список процессов

### Git
  gitui               TUI клиент для git (полноценный!)
  git status          Статус
  git add -p          Интерактивное добавление
  git diff            Изменения (с delta)

### Система
  btm                 Мониторинг (CPU, RAM, сеть)
  tokei               Статистика кода
  tldr <cmd>          Краткая справка
  hyperfine <cmd>     Бенчмарк команды

### История
  Ctrl+R              Поиск по истории (atuin)
