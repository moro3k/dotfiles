## 💡 Идея и юзкейс

### ASUS Zenbook 14X OLED (UX5400EG) + ScreenPad

Эта конфигурация создана для ноутбука **ASUS Zenbook** с уникальной особенностью — **сенсорным дисплеем встроенным в тачпад (ScreenPad)**.

Как это работает:
- **Основной OLED экран** — код, документация, браузер
- **ScreenPad (тачпад-дисплей)** — терминал с Claude Code

Это создаёт невероятный workflow:
- Пишешь промпт для Claude и **видишь что печатаешь прямо под руками**
- Не нужно переключать взгляд между клавиатурой и экраном
- Тачпад остаётся функциональным — можно скроллить, выделять текст
- Claude Code работает как **второй мозг**, всегда под рукой

```
┌─────────────────────────────────────────┐
│           Основной OLED экран           │
│    (Код, браузер, документация)         │
└─────────────────────────────────────────┘
         ┌─────────────────────┐
         │   ScreenPad/Тачпад  │
         │   ┌───────────────┐ │
         │   │ Claude Code   │ │
         │   │ > prompt_     │ │
         │   └───────────────┘ │
         └─────────────────────┘
              [Клавиатура]
```

## 🛠 Почему именно эти инструменты?

### WezTerm вместо Windows Terminal

| Критерий | Windows Terminal | WezTerm |
|----------|------------------|---------|
| GPU-ускорение | DirectX | WebGPU (кроссплатформенный) |
| Конфигурация | JSON (ограниченный) | **Lua (полноценный язык)** |
| Кастомизация статус-бара | ❌ Нет | ✅ Полная свобода |
| Интеграция с WSL | Хорошая | **Отличная** |
| Хоткеи | Базовые | **Любые комбинации** |
| Мультиплексор | ❌ Нет | ✅ Встроенный (но мы используем Zellij) |

**Главная причина:** возможность вывести **статус Claude Code прямо в tab bar** — контекст, токены, стоимость сессии в реальном времени.

### Zellij вместо tmux

| Критерий | tmux | Zellij |
|----------|------|--------|
| Язык | C | **Rust** |
| Скорость | Быстрый | **Очень быстрый** |
| Конфигурация | Сложная | **Интуитивная (KDL)** |
| UI из коробки | Минимальный | **Красивый, информативный** |
| Floating panes | Плагин | **Встроено** |
| Режимы | Один режим | **Модальный (как Vim)** |
| Mouse support | Частичный | **Полный** |

**Главная причина:** встроенные **floating panes** для popup-шпаргалки по F1, модальное управление и нативная поддержка мыши/тачпада.

## 📊 Статус-бар Claude Code

Уникальная фича — **real-time статус Claude Code** прямо в WezTerm:

```
ctx ██░░░░ 15% │ ↓8.0K ↑17.6K │ ⚡17.7K │ $1.45 │ +476/-436 │ Opus 4.5 │ 12:34
```

| Элемент | Описание |
|---------|----------|
| `ctx ██░░░░ 15%` | Использование контекста (из 200K токенов). Зелёный < 50%, жёлтый < 80%, красный > 80% |
| `↓8.0K` | Входящие токены (ваши сообщения + прочитанные файлы) |
| `↑17.6K` | Исходящие токены (ответы Claude) |
| `⚡17.7K` | Кэшированные токены (экономия на повторных запросах) |
| `$1.45` | Стоимость текущей сессии в USD |
| `+476/-436` | Строки кода: добавлено / удалено |
| `Opus 4.5` | Текущая модель |
| `12:34` | Время |

**Когда Claude Code не активен** — статус становится серым.

## ⌨️ Горячие клавиши

### Zellij — Мультиплексор

Zellij использует **модальное управление** (как Vim). Нажмите клавишу режима, затем действие, затем `Esc` для выхода.

#### Переключение режимов

| Клавиша | Режим | Описание |
|---------|-------|----------|
| `Ctrl+P` | **Pane** | Управление панелями |
| `Ctrl+T` | **Tab** | Управление вкладками |
| `Ctrl+N` | **Resize** | Изменение размеров |
| `Ctrl+S` | **Scroll** | Прокрутка буфера |
| `Ctrl+O` | **Session** | Управление сессией |
| `Ctrl+Q` | — | Выход из Zellij |
| `Esc` | — | Выход из текущего режима |

#### Режим панелей (`Ctrl+P`)

| Клавиша | Действие |
|---------|----------|
| `n` | Новая панель |
| `d` | Новая панель снизу |
| `r` | Новая панель справа |
| `x` | Закрыть панель |
| `f` | Fullscreen (развернуть/свернуть) |
| `w` | Toggle floating панели |
| `z` | Toggle рамки панелей |
| `h` / `←` | Фокус на панель слева |
| `j` / `↓` | Фокус на панель снизу |
| `k` / `↑` | Фокус на панель сверху |
| `l` / `→` | Фокус на панель справа |

#### Режим вкладок (`Ctrl+T`)

| Клавиша | Действие |
|---------|----------|
| `n` | Новая вкладка |
| `x` | Закрыть вкладку |
| `r` | Переименовать вкладку |
| `h` / `←` | Предыдущая вкладка |
| `l` / `→` | Следующая вкладка |
| `1`-`5` | Перейти к вкладке по номеру |

#### Режим ресайза (`Ctrl+N`)

| Клавиша | Действие |
|---------|----------|
| `h` / `←` | Уменьшить ширину влево |
| `j` / `↓` | Увеличить высоту вниз |
| `k` / `↑` | Уменьшить высоту вверх |
| `l` / `→` | Увеличить ширину вправо |
| `=` / `+` | Увеличить размер |
| `-` | Уменьшить размер |

#### Режим скролла (`Ctrl+S`)

| Клавиша | Действие |
|---------|----------|
| `j` / `↓` | Скролл вниз |
| `k` / `↑` | Скролл вверх |
| `Ctrl+D` | Полстраницы вниз |
| `Ctrl+U` | Полстраницы вверх |
| `g` | В начало буфера |
| `G` | В конец буфера |

#### Режим сессии (`Ctrl+O`)

| Клавиша | Действие |
|---------|----------|
| `d` | Отключиться (detach) — сессия сохраняется |
| `q` | Выйти из Zellij |

#### Специальные

| Клавиша | Действие |
|---------|----------|
| `F1` | 📋 **Шпаргалка** — floating popup со всеми хоткеями |

## 🚀 Rust CLI утилиты

Все стандартные утилиты заменены на быстрые аналоги на Rust:

| Команда | Утилита | Заменяет | Описание |
|---------|---------|----------|----------|
| `rg` | ripgrep | grep | Поиск в файлах (в 10x быстрее) |
| `fd` | fd-find | find | Поиск файлов |
| `bat` | bat | cat | Просмотр с подсветкой синтаксиса |
| `eza` | eza | ls | Список файлов с иконками |
| `dust` | du-dust | du | Размер директорий (визуально) |
| `procs` | procs | ps | Список процессов |
| `btm` | bottom | htop/top | Мониторинг системы |
| `sd` | sd | sed | Замена текста |
| `delta` | git-delta | diff | Красивый diff |
| `z` | zoxide | cd | Умная навигация |
| `gitui` | gitui | — | TUI для Git |
| `zellij` | zellij | tmux | Мультиплексор |
| `starship` | starship | — | Prompt |
| `atuin` | atuin | — | История команд |

### Алиасы

```bash
ll              # eza -la --icons --git
lt              # eza --tree --icons -L 2
la              # eza -a --icons
z <dir>         # zoxide — умный cd (помнит частые пути)
```

## 📥 Установка

### Требования

- Windows 11 + WSL2
- [WezTerm](https://wezfurlong.org/wezterm/)
- Ubuntu 24.04 в WSL
- [JetBrainsMono Nerd Font](https://www.nerdfonts.com/font-downloads)

### Шаги

```bash
# 1. Клонировать репозиторий
git clone https://github.com/moro3k/dotfiles ~/Projects/dotfiles
cd ~/Projects/dotfiles

# 2. Запустить установщик
chmod +x install.sh
./install.sh

# 3. Скопировать конфиг WezTerm (в Windows PowerShell)
copy \\wsl$\Ubuntu-24.04\home\ilnur\Projects\dotfiles\wezterm\.wezterm.lua $HOME\.wezterm.lua

# 4. Перезапустить WezTerm
```

### Ручная установка Rust утилит

```bash
# Установить Rust
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Установить утилиты
cargo install ripgrep fd-find bat eza du-dust procs bottom sd git-delta zoxide starship atuin gitui zellij
```
